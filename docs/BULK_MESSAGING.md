# 一斉送信・AI一括下書き（Bulk Messaging）設計メモ

1日に数十〜数百人を相手にするメイト/管理者の返信業務を高速化する機能。
「送信ボタンを押すのは人間」を原則に、下書きの一括生成と1人ずつの高速レビューを提供する。

## ユースケース
1. **同じ内容の一斉送信**: 「おはようございます！今日もいきましょう」などを複数ユーザーへ。`{名前}` プレースホルダで宛名を差し込み可能。
2. **AI下書きの一括生成**: 選択した各ユーザーの会話履歴・ピン留めメモ・チェックインを踏まえた**個別の**返信案をAIがまとめて生成。共通の指示（例:「週末の過ごし方を聞いてみて」）も添えられる。
3. **レビューキュー**: 生成された下書きを1人ずつ確認し、「そのまま送信」「追記して送信」「修正して送信」「スキップ」をワンタップ＋自動で次へ。まとめて送る場合は「残りすべて送信」。

## 設計判断
| 論点 | 判断 | 理由 |
|---|---|---|
| データモデル | **新テーブルなし** | AI下書きは既存 `ai_draft_requests`/`ai_drafts` に永続化される。送信は既存 `messages`。キュー自体は画面内の一時状態で十分（送信済みはDBに残る） |
| 送信エンジン | **クライアント駆動で既存 `sendMessage` を1人ずつ呼ぶ**（並列2） | 権限（`canSendMessage`＝担当チェック・Shadow拒否）・監査ログ・LINE push を完全再利用。サーバーレスのタイムアウトを回避し、部分成功が見える・失敗だけ再試行できる |
| AI生成エンジン | クライアント駆動で既存 `generateAiDrafts` を1人ずつ（並列2・逐次表示） | 100人分を1リクエストで生成するとタイムアウト。並列2で先に終わった人からレビュー開始できる |
| AI回数制限 | 既存の1ユーザー/日 `AI_DRAFT_DAILY_LIMIT`（現在5回）をそのまま適用 | 一括生成は1ユーザーにつき1回消費 |
| 共通指示 | `generateAiDrafts` に任意 `instruction`（200字）を追加 | プロンプト末尾に「メイトからの指示」として反映。後方互換 |
| 対象選択 | 受信トレイの**選択モード**（チェックボックス）＋表示中を全選択 | 既存フィルタ（今日未送信/未返信/タグ等）で絞ってから全選択、が主動線 |
| 権限 | 全ロール利用可。実際に送れる相手は `sendMessage` 側で担当チェック | UIで新しい権限を発明しない |

## 主要フロー
```
受信トレイ → [まとめて送信] → 行をタップで選択（フィルタ＋全選択）
  → [同じ内容を送信]  → 本文作成（テンプレ/{名前}） → レビューキュー
  → [AIで下書き生成] → 共通指示(任意)          → レビューキュー（生成が終わった人から順に）
レビューキュー: 前へ/次へ・編集・[送信して次へ](⌘Enter)・[スキップ]・[残りすべて送信]（確認あり）
完了ビュー: 送信/スキップ/失敗の集計 → 失敗は個別再試行、スキップは戻って確認
```

## 追加機能（第2弾）
- **ワンタップ導線**: ダッシュボード/受信トレイの警告バナーに「まとめて送る」。`/inbox?reply=not_sent_today&bulkSelect=1` で選択モード＋表示中全選択で開始（URLの `bulkSelect` は消費後に除去）
- **予約送信**: レビューキューの「一括送信/予約」からプリセット時刻（今日19時/明日8時/明日12時）で予約。`scheduled_messages` に登録し、cron（`/api/jobs/scheduled-messages`・10分間隔）が期日到来分をLINE送信して `messages` に記録。楽観ロックで多重実行に耐え、送信時点で未契約/ブロック済みはスキップ。ツールバーの「🕒予約N件」から取り消し可能
- **テンプレ学習**: 選択のたびに `usage_count` を加算し、よく使うテンプレを上に表示
- **変数**: `{名前}` に加えて `{プラン}` `{最終チェックイン}` を差し込み可能
- **一括操作（Admin/SV）**: 下部バーの[⋯]から「タグを追加」（重複追加なし）・「担当メイトを変更」（既存 assignCast を順次実行＝キャパ検証・監査・LINE通知を再利用）

## 実装ファイル
- `components/inbox/BulkSendController.tsx` — 選択状態・ツールバー・下部アクションバー・モーダル配線
- `components/inbox/BulkComposeModal.tsx` — 同一本文の作成（テンプレ再利用・`{名前}`）
- `components/inbox/BulkAiModal.tsx` — AI一括生成の開始（共通指示）
- `components/inbox/BulkReviewPanel.tsx` — レビューキュー本体（生成/送信エンジン内蔵・portal全画面）
- `components/inbox/InboxList.tsx` — 選択モード対応（checkbox・タップでトグル）
- `actions/ai.ts` / `schemas/ai.ts` — `instruction` 対応
